É uma [abertura de flanco](Xadrez_Aberturas_de_flanco.md) incomum no [xadrez](../../../../sementes/2024/07/06/Xadrez.md) e por isso é categorizada como uma [Abertura irregular](../../../../sementes/2024/07/06/Xadrez_Aberturas_irregulares.md) com código [ECO](../../../../sementes/2024/07/07/Encyclopaedia_of_Chess_Openings.md) `A0` na lista de [aberturas de xadrez](Xadrez_Aberturas.md). Pode transpor para uma [siciliana reversa](Xadrez_Abertura_Inglesa.md).
