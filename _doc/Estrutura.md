A estrutura de arquivos não precisa ser fixo e padronizado para todo mundo. O importante é entender qual a ideia geral para se criar uma estrutura de pastas que faça sentido para permitir que você gaste menos tempo pensando em qual pasta cada documento deveria ir, além de entender o minimo das questões técnicas associados para a gestão dos arquivos (para que o [Software](../api/sementes/2024/07/02/Software.md) continue eficiente e não te atrapalhe).
